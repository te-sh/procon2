$ N $ 以下の素数は列挙しておく.

数値 $ x $ で手番が回ってきたときに勝てるかどうかを $ B(x) $ とする. $ B(0) = B(1) = \text{true} $ である.

このとき, すべての素数 $ p \ (p \leq x) $ について $ B(x-p) = \text{true} $ ならば $ B(x) = \text{false} $ であり, そうでなければ $ B(x) = \text{true} $ となる. これを順に計算していき, $ B(N) $ を見る.
